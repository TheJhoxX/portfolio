Se trata de un portfolio personal que he realizado como "carta de presentación". En él enseño algunos proyectos que se pueden considerar interesantes (varios de ellos son trabajos realizados durante mi formación académica)
Para iniciar el servidor de desarrollo se deberá ejecutar: 

npm run dev 

En la carpeta del repositorio. 
En caso de querer visitar el portfolio desplegado acceder al enlace:

https://portfolio-two-tau-88.vercel.app/

El portfolio ha sido realizado empleado Next.js + TailwindCSS además de NextUI como librería de componentes

**Actualmente solo cuenta con un tema oscuro, pronto contará con un tema claro para que pueda poder tener una interfaz agradable para todos los usuarios**
